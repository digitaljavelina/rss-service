         ```typescript
         interface PreviewRequest {
           url: string;

             description?: string;
             date?: string;
           };
         }
         ```

            ```typescript
            {
              success: true,
              items: items.map(i => ({
                title: i.title,
                link: i.link,
                description: i.description,
                pubDate: i.pubDate?.toISOString()
              })),
              metadata: {
                pageTitle: pageTitle,
                itemCount: items.length,
                fetchedAt: new Date().toISOString()
              }
            }
            ```

<task type="auto">
  <name>Task 2: Create feed CRUD API and mount routes</name>
  <files>
    src/routes/api/feeds.ts
    src/routes/index.ts
  </files>
  <action>
    1. Create `src/routes/api/feeds.ts`:
       - Import Router from express
       - Import { nanoid } from 'nanoid'
       - Import slugify from 'slugify'
       - Import { createHash } from 'crypto'
       - Import { supabase } from '../../db/index.js'
       - Import { fetchPage } from '../../services/page-fetcher.js'
       - Import { extractItems } from '../../services/content-extractor.js'
       - Import { parseDate } from '../../services/date-parser.js'
       - Import types from '../../types/feed.js'

       - Export `feedsApiRouter = Router()`

       - POST '/' - Create new feed:
         Request body:
         ```typescript
         {
           name: string;
           url: string;
           selectors: FeedSelectors;
         }
         ```

         a. Validate: name, url, selectors.item, selectors.title required
            Return 400 if missing

         b. Generate feed ID: nanoid()
         c. Generate slug: slugify(name, { lower: true, strict: true })
            If slug exists in DB, append nanoid(6) to make unique

         d. Insert feed into database:
            ```typescript
            await supabase.from('feeds').insert({
              id: feedId,
              slug: slug,
              name: name,
              url: url,
              selectors: JSON.stringify(selectors),
              item_limit: 100
            })
            ```

         e. Fetch page and extract items:
            - fetchPage(url)
            - extractItems(html, url, selectors)
            - For each item, generate GUID: createHash('sha256').update(`${feedId}:${item.title}:${item.link}`).digest('hex').substring(0, 16)

         f. Insert items into database:
            ```typescript
            await supabase.from('items').insert(
              items.map(item => ({
                id: nanoid(),
                feed_id: feedId,
                title: item.title,
                link: item.link,
                description: item.description || null,
                pub_date: item.pubDate?.toISOString() || new Date().toISOString(),
                guid: generateGuid(feedId, item)
              }))
            )
            ```

         g. Return 201 with:
            ```typescript
            {
              id: feedId,
              slug: slug,
              name: name,
              feedUrl: `${process.env.BASE_URL || 'http://localhost:3000'}/feeds/${slug}`,
              itemCount: items.length
            }
            ```

       - GET '/' - List all feeds:
         Query feeds table, return array of { id, slug, name, url, itemCount, createdAt, updatedAt }
         Include item count via separate query or join

       - GET '/:id' - Get single feed:
         Find by id or slug, return feed details with selectors

    2. Update `src/routes/index.ts`:
       - Add import: `import { previewRouter } from './api/preview.js'`
       - Add import: `import { feedsApiRouter } from './api/feeds.js'`
       - Mount BEFORE existing routes:
         ```typescript
         router.use('/api/preview', previewRouter);
         router.use('/api/feeds', feedsApiRouter);
         ```

    Note: Existing feedsRouter at /feeds serves XML - keep it separate from API.
