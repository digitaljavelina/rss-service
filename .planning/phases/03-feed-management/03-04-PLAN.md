**Test Dashboard (MGMT-01, UI-02):**
1. Navigate to /feeds (or click "My Feeds" in sidebar)
2. Verify table shows: Feed Name, Status badge, Item Count, Last Updated, Actions
3. If you have feeds, verify they appear in the table

**Test Refresh (MGMT-04 - already exists, verify in dashboard):**
1. Click "Refresh" on any feed row
2. Verify loading state appears
3. Verify success feedback (timestamp updates, new items if any)

**Test Edit (MGMT-02):**
1. Click feed name or Edit link in dashboard
2. Verify edit page shows current configuration
3. Change the feed name
4. Click Save, verify success message
5. Go back to dashboard, verify name changed

**Test Delete (MGMT-03, UI-03):**
1. Create a test feed first (or use existing)
2. In dashboard, click Delete on a feed
3. Verify confirmation modal appears with feed name
4. Click Cancel, verify modal closes and feed remains
5. Click Delete again, confirm deletion
6. Verify feed removed from table
7. Verify feed's RSS URL returns 404

**Test Export (MGMT-06):**
1. Click Export on any feed
2. Verify JSON file downloads
3. Open file, verify it contains: name, url, selectors, exportedAt

**Test Import (MGMT-07):**
1. Use the exported JSON file from above
2. Click "Import Feed" button
3. Select the JSON file
4. Verify new feed appears in dashboard
5. Verify imported feed works (can access RSS URL)

**Test Error States (UI-03):**
1. Try importing invalid file (non-JSON)
2. Verify clear error message
3. Try importing JSON with invalid URL
4. Verify clear error message
