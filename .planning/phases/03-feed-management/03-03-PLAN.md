<task type="auto">
  <name>Task 2: Add export/import handlers to dashboard.js</name>
  <files>public/js/dashboard.js</files>
  <action>
Update public/js/dashboard.js to add export and import functionality:

1. Add DOM references:
   - importFileInput (input#import-file)
   - importBtn (button that triggers file input)
   - importError (div#import-error)

2. Update createFeedRow() to add Export button:
   - Add Export button with data-action="export", data-feed-slug, data-feed-name
   - Place between Refresh and Edit link in actions cell

3. Add exportFeed(slug, name) function:
   - Fetch feed details: GET /api/feeds/{slug}
   - Create config object: { name, url, selectors, itemLimit, exportedAt: new Date().toISOString() }
   - Create JSON string: JSON.stringify(config, null, 2)
   - Create Blob: new Blob([json], { type: 'application/json' })
   - Create download URL: URL.createObjectURL(blob)
   - Create temp anchor, set href and download attribute (filename: {slug}-config.json)
   - Trigger click on anchor
   - Revoke URL immediately after: URL.revokeObjectURL(url)

4. Add event listener for export action:
   - In the document click handler, add case for data-action="export"
   - Call exportFeed(e.target.dataset.feedSlug, e.target.dataset.feedName)

5. Add validateFeedConfig(config) function:
   - Check config.name exists and is string
   - Check config.url exists and is string
   - Try new URL(config.url) - return false if invalid
   - Return true if all checks pass

6. Add importFeed(config) async function:
   - POST to /api/feeds with { name: config.name, url: config.url }
   - If selectors in config, include them in request (optional)
   - On success: reload feed list, show success message
   - On error: show error message

7. Add import file input handler:
   - importFileInput.addEventListener('change', ...)
   - Get file from e.target.files[0]
   - Create FileReader, read file as text
   - reader.onload: parse JSON, validate, call importFeed()
   - Handle parse errors with clear message
   - Handle validation errors with clear message
   - Reset file input value after processing (allows re-selecting same file)

8. Add click handler for Import button:
   - importBtn.onclick triggers importFileInput.click()

9. showImportError(message) helper:
   - Show import-error div with message

10. hideImportError() helper:
    - Hide import-error div

Following patterns from RESEARCH.md for safe file handling and memory cleanup.
