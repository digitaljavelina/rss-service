# Plan 06-03: Reddit RSS Integration Service

## Goal
Create a Reddit service that fetches built-in RSS feeds from subreddits and user pages, parsing them into standard feed items.

## Tasks

### 1. Reddit Service
File: `src/services/reddit.ts` (new)

Functions:
- `parseRedditUrl(url: string)` — Extract subreddit name, username, and sort mode from Reddit URLs
- `buildRedditRssUrl(parsed: RedditUrlInfo)` — Construct the RSS feed URL (e.g., `https://www.reddit.com/r/programming.rss`)
- `fetchRedditRss(rssUrl: string)` — Fetch RSS XML with proper User-Agent header
- `parseRedditRss(xml: string, sourceUrl: string)` — Parse Atom/RSS XML into ExtractedItem[]
- `redditToFeedItems(entries: RedditEntry[])` — Map parsed entries to ExtractedItem[]

Key design:
- Use `cheerio` to parse Atom XML (consistent with rest of codebase)
- Set User-Agent: `RSS-Service/1.0` to avoid 429 errors
- Handle both Atom (`<entry>`) and RSS (`<item>`) formats
- Strip HTML from description content

### 2. Reddit Preview Support
File: `src/routes/api/preview.ts`

Update POST /api/preview to detect Reddit URLs:
- If URL matches reddit.com patterns, use Reddit service
- No API key needed — direct RSS fetch
- Return items + metadata (subreddit name, post count)

### 3. Reddit Feed Creation Support
File: `src/routes/api/feeds.ts`

Update POST /api/feeds:
- If URL is Reddit, set `feed_type: 'reddit'`
- Store `platform_config: { subreddit, sort, feedUrl }`
- Store `selectors: '{}'` (not used for Reddit feeds)
- Items created from RSS response

### 4. Reddit Refresh Support
File: `src/routes/api/feeds.ts`

Update POST /api/feeds/:id/refresh:
- If `feed_type === 'reddit'`, use Reddit service instead of web scraping
- Re-fetch RSS, parse, deduplicate, and insert new items

## Verification
- [ ] `parseRedditUrl` handles /r/sub, /r/sub/new, /u/user URL patterns
- [ ] `buildRedditRssUrl` generates correct .rss URLs
- [ ] `fetchRedditRss` returns RSS XML from Reddit
- [ ] `parseRedditRss` extracts title, link, description, date from Atom feed
- [ ] Preview API returns Reddit posts for a subreddit URL
- [ ] Feed creation stores feed_type='reddit' with platform_config
- [ ] Manual refresh fetches new posts via RSS
