  <how-to-verify>
    **Start the application:**
    ```bash
    npm run dev
    ```

    **Test 1: Create Feed Flow (CORE-01, CORE-02, CORE-03)**
    1. Open http://localhost:3000/create
    2. Enter feed name: "HN Test"
    3. Enter URL: https://news.ycombinator.com
    4. Enter selectors:
       - Item: .athing
       - Title: .titleline > a
       - Link: .titleline > a
    5. Click "Preview" button
    6. Verify: Preview section shows extracted HN stories with titles and links
    7. Click "Save Feed" button
    8. Verify: Success message appears with feed URL

    **Test 2: Feed Serving - RSS (CORE-04)**
    1. Click the feed URL link from Test 1
    2. Verify: Browser shows valid RSS XML with items
    3. Or use curl to check:
       ```bash
       curl -s http://localhost:3000/feeds/YOUR_SLUG | head -30
       ```
       Should show RSS 2.0 XML with <item> elements

    **Test 3: Feed Serving - Atom (CORE-05)**
    Request feed with Accept header for Atom format:
    ```bash
    curl -H "Accept: application/atom+xml" http://localhost:3000/feeds/YOUR_SLUG | head -30
    ```
    Verify: Returns Atom XML (should contain `<feed xmlns="http://www.w3.org/2005/Atom">`
    and `<entry>` elements instead of `<item>`)

    **Test 4: Manual Refresh (MGMT-04)**
    1. Note the current item count in your feed
    2. Run in terminal:
       ```bash
       curl -X POST http://localhost:3000/api/feeds/YOUR_SLUG/refresh
       ```
    3. Verify: Response shows { success: true, newItems: N, totalItems: M }
    4. Refresh the feed XML page
    5. Verify: Feed content is current (may have new items)

    **Test 5: Export Feed (OUT-03)**
    1. Download RSS export:
       ```bash
       curl -o test-feed.xml http://localhost:3000/api/feeds/YOUR_SLUG/export
       ```
    2. Verify: File downloads with Content-Disposition header
    3. Check content: `head test-feed.xml` shows valid RSS XML
    4. Download Atom export:
       ```bash
       curl -o test-feed-atom.xml "http://localhost:3000/api/feeds/YOUR_SLUG/export?format=atom"
       ```
    5. Verify: `head test-feed-atom.xml` shows valid Atom XML

    **Test 6: Error Handling**
    1. Go to /create, leave fields empty, click Preview
    2. Verify: Validation error shown
    3. Enter URL: https://httpstat.us/404
    4. Click Preview
    5. Verify: Fetch error shown gracefully
