   - Pattern: /feeds/{slug}/edit

3. Track original values:
   - Store originalUrl when feed loads (to detect URL changes)

4. loadFeed() function:
   - fetch('/api/feeds/{slug}')
   - On success: populate form fields (name, url, item limit)
   - Store feed.id for save operation
   - Store originalUrl = feed.url for comparison
   - On error: show error message, disable save button

5. setLoading(btn, loading) helper:
   - Same pattern as create-feed.js
   - Toggle loading spinner and disabled state

6. validateForm() function:
   - Name required, 1-100 chars
   - URL must be valid URL format
   - Item limit must be positive number 1-1000
   - Return array of error strings

7. showError(message) function:
   - Show error section with message
   - Hide success section

8. showSuccess(message) function:
   - Show success section with message
   - Hide error section

9. hideMessages() function:
   - Hide both error and success sections

10. Save button click handler (btnSave.onclick):
    - hideMessages()
    - Run validation, show errors if any
    - setLoading(btnSave, true)
    - Determine if URL changed: currentUrl !== originalUrl
    - PUT to /api/feeds/{feedId} with { name, url, itemLimit }
    - On success:
      - If response.urlChanged: showSuccess('Feed updated. URL changed - selectors re-detected.')
      - Else: showSuccess('Feed updated successfully')
      - Update originalUrl to new URL value
    - On error: showError(error message from response)
    - setLoading(btnSave, false)

11. Copy feed URL functionality:
    - Add copy button next to feed URL
    - On click: navigator.clipboard.writeText(feedUrl)
    - Show brief "Copied!" feedback

12. Call loadFeed() on DOMContentLoaded
